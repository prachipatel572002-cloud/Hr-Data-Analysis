# 📊 HR Data Analysis Dashboard (Power BI)

## 📌 Project Overview
This project focuses on analyzing HR data to uncover key insights related to employee performance, attrition, demographics, and overall workforce trends. The dashboard is built using Power BI to support data-driven decision-making.

---

## 🎯 Objectives
- Analyze employee attrition and retention patterns  
- Understand workforce distribution (age, gender, department)  
- Identify key factors affecting employee performance  
- Provide actionable insights for HR decision-making  

---

## 🛠️ Tools & Technologies Used
- Power BI (Data Visualization & Dashboard)
- Microsoft Excel (Data Source)
- Power Query (Data Cleaning & Transformation)
- DAX (Data Analysis Expressions)

---

## 📂 Dataset Information
- Dataset: HR Data (Excel file)
- Includes:
  - Employee demographics  
  - Job roles & departments  
  - Salary & performance metrics  
  - Attrition status  

---

## 📊 Key Insights
- Identified departments with highest attrition rates  
- Analyzed salary vs attrition relationship  
- Visualized employee distribution across roles and age groups  
- Highlighted trends in employee retention  

---

## 📈 Dashboard Features
- Interactive filters (Department, Gender, Job Role)
- KPI cards (Total Employees, Attrition Rate, Average Salary)
- Charts:
  - Bar charts
  - Pie charts
  - Matrix

---

## 🚀 How to Use
1. Download the `.pbix` file from this repository  
2. Open it using Power BI Desktop  
3. Explore the dashboard using filters and visuals
